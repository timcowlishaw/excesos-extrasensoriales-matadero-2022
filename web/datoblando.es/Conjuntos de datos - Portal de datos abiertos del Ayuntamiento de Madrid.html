OUTLIERS (VALOR ATÍPICO):

  1. es una observación anormal y extrema en una muestra estadística de datos
  2. son valores excepcionalmente lejanos del centro. Es una observación numéricamente distante del resto de los datos.
  3. Las estadísticas derivadas de los conjuntos de datos que incluyen valores atípicos serán frecuentemente engañosas.
  4. una observación dentro de una muestra de datos que no es consistente con el resto.

DATO:

  1. Información sobre algo concreto que permite su conocimiento exacto o sirve para deducir las consecuencias derivadas de un hecho.
  2. Información dispuesta de manera adecuada para su tratamiento por una computadora. Cifra, letra o palabra que se suministra a la computadora como entrada y la máquina almacena en un determinado formato.
